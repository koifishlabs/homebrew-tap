# koifishlabs/homebrew-tap

Homebrew tap for [Fusuma](https://fusuma.dev) — a Mac IDE for running coding agents (Claude Code, Codex, …) in parallel.

## Install

```sh
brew install --cask koifishlabs/tap/fusuma
```

Or in two steps:

```sh
brew tap koifishlabs/tap
brew install --cask fusuma
```

## Upgrade

```sh
brew upgrade --cask fusuma
```

Fusuma also ships its own in-app auto-updater, so `brew upgrade` is optional.

## Uninstall

```sh
brew uninstall --cask fusuma
brew untap koifishlabs/tap
```
